AFTER Trigger

AFTER triggers are fired after the triggering action is completed For example, if AFTER trigger is associated with INSERT command then it is fired after the row is inserted into the table.

EXAMPLE

CREATE A TRIGGER FOR THE EMP TABLE, WHICH MAKES THE ENTRY IN NAME COLUMN IN UPPERCASE 
CREATE OR REPLACE TRIGGER log_changes_trigger
AFTER INSERT OR UPDATE OR DELETE ON product
FOR EACH ROW
BEGIN
IF INSERTING THEN
            DBMS_OUTPUT.PUT_LINE('New row inserted: ' || :NEW.product_id);
ELSIF UPDATING THEN
            DBMS_OUTPUT.PUT_LINE('Row updated: ' || :OLD.product_id || ' -> ' || :NEW.product_id);
ELSIF DELETING THEN
            DBMS_OUTPUT.PUT_LINE('Row deleted: ' || :OLD.product_id);
END IF;
    END;
