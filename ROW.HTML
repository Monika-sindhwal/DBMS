ROW-LEVEL Trigger

A row-level trigger is fired each time the table is affected by the triggering statement, ie it is fired once for each row that is affected by the triggering statement

For example, if an UPDATE statement updates multiple rows of a table, a row trigger fired once for each row affected by the UPDATE statement. If a triggering statement affeCts no row's, a row trigger is not executed at all.

Row-level triggers are useful if the code in the trigger action depends on data provided by the triggering statement or rows that are affected. Row-level triggers are implemented by using the option FOR EACH ROW in CREATE TRIGGER statement

EXAMPLE 

CREATE OR REPLACE TRIGGER ROW_LEVEL
BEFORE UPDATE ON EMP
FOR EACH ROW
DECLARE 
 X VARCHAR(30):='ROW LEVEL TRIGGER FIRED';
 BEGIN
 DBMS_OUTPUT.PUT_LINE(X);
 END;
UPDATE EMP

SET SAL=11000

WHERE DEPT=30;

ROW LEVEL TRIGGER FIRED;
ROW LEVEL TRIGGER FIRED;
(2 rows updated.)
