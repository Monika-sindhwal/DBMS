NESTED QUERY

Nested query is a query within another query. SQL has an ability to nest queries within ne another. A subquery is a SELECT statement that is nested within another SELECT atement and which returns intermediate results. The result of subquery or inner query is ynamically substituted in the condition of outer query. SQL first evaluates the inner query r subquery within the WHERE clause. The result of inner query is then substituted in the ondition of the outer query.
Most of the time, a subquery is used when you know how to search for a value using a SELECT statement, but do not know the exact value.

List the employees belonging to the department of MILLER from "EMP" table.

Here, we do not know the department to which MILLER belongs. So, we have to determine the department of MILLER first and then use that department number to find out the other employees of that department. 
SELECT DEPTNO FROM EMP
WHERE ENAME = 'MILLER' ;
OUTPUT:
DEPTNO
10

After finding the department number of MILLER, the next query to find out the other employees of that department will be: 

SELECT ENAME FROM EMP
WHERE DEPTNO = 10;

CLARK
KING
MILLER
