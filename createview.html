CREATE VIEW


SYNTAX: 

CREATE VIEW view_name
AS
SELECT column_list
FROM table_name [WHERE CONDITION];
In this syntax:
view_name - is the name of the view;
The SELECT statement is used to define the columns and the rows that you want to display in the view.

EXAMPLE


Create a view for the employees belonging to department number 20 of EMP table. 
CREATE VIEW V1
AS
SELECT * FROM EMP
WHERE DEPTNO = 20;

After creating a view, it can be queried just like querying a table.

SELECT * FROM V1;
